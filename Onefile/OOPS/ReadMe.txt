1. Classes and Objects:
Class Definition:

A class is a blueprint for creating objects.
Define a class using the class keyword.
Objects:

Objects are instances of a class.
Create objects by calling the class as if it were a function.
2. Attributes and Methods:
Attributes:

Properties that store data within a class.
Accessed using dot notation (e.g., object.attribute).
Methods:

Functions defined within a class.
Accessed using dot notation (e.g., object.method()).
